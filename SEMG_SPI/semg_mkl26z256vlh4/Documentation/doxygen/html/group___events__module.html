void DMAT_M_SPI_RX_OnComplete ( LDD_TUserData *  UserDataPtr)

Called at the end of a DMA transfer. If the Half complete property in initialization section is anabled, this event is also called when current major iteration count reaches the halfway point. See SetEventMask() and GetEventMask() methods. This event is enabled only if Interrupts property in Channel select section is enabled.

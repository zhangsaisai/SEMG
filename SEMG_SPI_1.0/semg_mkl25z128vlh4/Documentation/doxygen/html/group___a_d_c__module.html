void EnableADCSPI ( EADCFlag  adcFlag)

This method pull the CS signal down to enable corresonding ADC's SPI. Each time only one ADC's SPI is available.

This method reads ADC's data when ADC's is ready to be read. This method is called in function EINT_AD_NOT_DRDY0_OnInterrupt or EINT_AD_NOT_DRDY0_OnInterrupt, file Events.c.

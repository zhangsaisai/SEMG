Detailed Description

A structure of channel data from ADC.

 |  Head   |   LOFF_STATP    |   LOFF_STATN    |GPIO[4:7]|                  CH[1-8] Data                 |
 ---------------------------------------------------------------------------------------------------------
 | 1 1 0 0 | x x x x x x x x | x x x x x x x x | x x x x | CH1 | CH2 | CH3 | CH4 | CH5 | CH6 | CH7 | CH8 |
 ---------------------------------------------------------------------------------------------------------
 |                        3Bytes                         |                  8*2=16Bytes                  |
 ---------------------------------------------------------------------------------------------------------

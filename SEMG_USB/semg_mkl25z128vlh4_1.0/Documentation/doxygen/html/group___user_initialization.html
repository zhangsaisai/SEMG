Detailed Description

This module contains user initialization of data, on-chip and peripheral devices.

The followings will be initialized:

  • User data
  • On-chip devices
    • SPI0, SPI1
    • SPI0_TX_DMA, SPI0_RX_DMA, SPI1_TX_DMA, SPI1_RX_DMA
    • DMA
    • UART0, UART2
    • PTA2
    • PTB1, PTB19
    • PTC1, PTC2
    • PTD4
    • PTE18, PTE19
  • Peripheral devices
    • ADC
